Date and Time. Promises.

Color switcher

Wrote a script that, after pressing the "Start" button, changes the background color of the to a random value once per second, using the inline style. Pressing the "Stop" button stops the background color change.

Countdown timer

A timer script has been written that counts down to a certain date. Such a timer can be used in blogs and online stores, event registration pages, during maintenance, etc. The flatpickr library was used to allow the user to cross-browser select the final date and time in one interface element.

Promise generator

A script has been written that, at the time of form submission, calls the createPromise(position, delay) function as many times as entered in the amount field. During each call, pass it the number of the position being created and the delay, taking into account the first delay entered by the user and the step.
